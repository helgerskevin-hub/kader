HOW TO INSTALL
- Web/desktop app: download the .skill package, then Settings > Skills > upload.
- Claude Code (CLI/desktop): copy the skill folder into Settings > Skills >
  Open Skills Folder, then restart.

Note: These skills help you ANALYZE and structure decisions. None of them
automate live trades on eToro. Crypto and copy trading carry real risk of
capital loss — analysis is a decision aid, not financial advice.
