# Kader

**Structuur in crypto.**

Persoonlijke crypto copy-trading app. Analyseert gratis marktdata en geeft concrete technische signalen, inclusief entry, stop-loss en take-profit. Geen hype, gewoon data.

> Geen financieel advies. Kader levert technische signalen ter ondersteuning van eigen onderzoek. Handel nooit met geld dat je niet kunt missen.

---

## Wat het doet

- Analyseert dagelijks een vaste set coins via Binance en CoinGecko (publiek, geen API-key)
- Berekent entry, stop-loss (ATR-gebaseerd) en take-profit (minimaal 1:2 R/R)
- Beoordeelt eToro Popular Investors op consistentie, drawdown en spreiding
- Geeft pushmeldingen bij actie nodig (take-profit verhogen of positie verkleinen)

## Platform

Android: React Native + Expo (`app/`)
